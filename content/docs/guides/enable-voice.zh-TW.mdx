---
title: 啟用語音互動
description: 設定與 Jarvis 的語音互動。
---

本指南將引導你在 CoreLayer 中啟用語音管線。

## 前置條件

- CoreLayer 已安裝
- 已授予應用程式麥克風存取權限

## 步驟 1：啟用語音

開啟 **設定 → 語音** 並切換**啟用語音管線**。

## 步驟 2：設定喚醒詞

啟用喚醒詞偵測以透過語音啟動 Jarvis：

```json
{
  "voice": {
    "wakeWord": {
      "enabled": true,
      "keyword": "jarvis"
    }
  }
}
```

喚醒詞使用 Picovoice Porcupine 在本地運行 — 無需雲端處理。

## 步驟 3：選擇 ASR 供應商

| 供應商 | 設定 |
|----------|-------|
| **Web Speech API** | 無需設定 — 開箱即用 |
| **Groq Whisper** | 需要在設定 → 模型中設定 Groq API 金鑰 |

建議入門使用 Web Speech API。Groq Whisper 提供更高的準確度。

## 步驟 4：測試語音

說出喚醒詞（「Jarvis」）或按一下控制中心的麥克風按鈕。然後說出你的請求：

```text
我今天有什麼任務？
```

Jarvis 將處理你的語音、產生回應，並使用 TTS 播放出來。

## 語音覆蓋層

當語音啟動時，浮動覆蓋層會顯示：

- **聆聽** — 波形動畫，青色強調
- **處理** — 旋轉指示器，紫色強調
- **播放中** — 播放進度，翡翠色強調

## 打斷插話

你可以在 Jarvis 回應期間再次說話來打斷。管線將停止目前的回應並處理你的新輸入。

## 疑難排解

- **喚醒詞未偵測**：檢查麥克風權限並減少背景噪音
- **語音未辨識**：嘗試說得更清楚或切換到 Groq Whisper
- **無音訊輸出**：檢查系統音訊和 TTS 設定
