---
title: 工具註冊表
description: 統一的工具註冊、發現和路由。
---

工具註冊表是所有可供 Jarvis 使用的工具的中央目錄。它聚合來自多個來源的工具，並透過統一介面呈現它們。

## 工具來源

| 來源 | 說明 |
|--------|-------------|
| **原生模組** | 內建的 CoreLayer 工具（任務管理、閱讀清單、回顧） |
| **MCP 伺服器** | 來自已連接 MCP 伺服器的工具 |
| **技能** | 自訂腳本工作流程 |
| **REST 轉接器** | 來自 REST API 端點的工具 |

## 運作方式

```text
工具來源註冊它們的工具
  → 工具註冊表編目所有工具
  → Jarvis 為請求查詢可用工具
  → 工具註冊表傳回符合的工具
  → Jarvis 選擇並呼叫適當的工具
  → 執行前經權限守護檢查
```

## 工具中繼資料

註冊表中的每個工具包含：

- **名稱** — 唯一識別碼
- **說明** — 工具的功能
- **參數** — 輸入結構（JSON Schema）
- **來源** — 哪個供應商擁有此工具
- **風險等級** — 讀取、寫入、外部、破壞性

## 查詢工具

Jarvis 可以查詢註冊表以按以下方式尋找工具：

- **功能** — 「尋找可以讀取檔案的工具」
- **來源** — 「顯示所有 MCP 工具」
- **名稱** — 「使用 filesystem:read 工具」
