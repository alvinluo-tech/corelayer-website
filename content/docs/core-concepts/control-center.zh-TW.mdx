---
title: 控制中心
description: 管理模型、工具、權限和系統健康狀態的中央介面。
---

控制中心是 CoreLayer 的主要介面。它提供了一個統一的介面來管理系統的所有方面。

## 面板

| 面板 | 用途 |
|-------|---------|
| **聊天** | 與 Jarvis 的主要對話 |
| **模型** | 設定和監控模型供應商 |
| **MCP** | 管理 MCP 伺服器連接 |
| **工具** | 檢視已註冊的工具及其狀態 |
| **權限** | 核准佇列和稽核日誌 |
| **語音** | 語音管線狀態和設定 |
| **設定** | 系統層級設定 |

## 指令面板

按 **Alt+Space** 開啟指令面板以進行快速操作：

- 搜尋對話
- 直接執行工具
- 在面板之間切換
- 快速設定

## 狀態列

控制中心底部顯示：

- 守護程式健康狀態
- 已連接的 MCP 伺服器數量
- 活動模型供應商
- 語音管線狀態
- 待核准數量

## 佈局

控制中心使用側邊欄佈局：

- **左側側邊欄** — 面板之間的導航
- **主區域** — 活動面板內容
- **右側面板** — 上下文相關的詳細資訊（工具資訊、核准詳情）
