title: 工具注册
description: 统一的工具注册、发现和路由。
---

工具注册是 Jarvis 可用的所有工具的中心目录。它聚合来自多个来源的工具，并通过统一接口呈现。

## 工具来源

| 来源 | 描述 |
|------|------|
| **原生模块** | 内置 CoreLayer 工具（任务管理、阅读列表、回顾） |
| **MCP 服务器** | 来自已连接 MCP 服务器的工具 |
| **技能** | 自定义脚本化工作流 |
| **REST 适配器** | 来自 REST API 端点的工具 |

## 工作原理

```text
工具来源注册其工具
  → 工具注册编目所有工具
  → Jarvis 为请求查询可用工具
  → 工具注册返回匹配的工具
  → Jarvis 选择并调用适当的工具
  → 权限守卫在执行前检查
```

## 工具元数据

注册中的每个工具包含：

- **名称** — 唯一标识符
- **描述** — 工具的功能
- **参数** — 输入模式（JSON Schema）
- **来源** — 哪个提供商拥有此工具
- **风险级别** — 读取、写入、外部、破坏性

## 查询工具

Jarvis 可以查询注册中心，通过以下方式查找工具：

- **能力** — "查找可以读取文件的工具"
- **来源** — "显示所有 MCP 工具"
- **名称** — "使用 filesystem:read 工具"
