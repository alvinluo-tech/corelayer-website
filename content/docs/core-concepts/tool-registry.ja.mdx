---
title: ツールレジストリ
description: 統一されたツール登録、ディスカバリー、ルーティング。
---

ツールレジストリは、Jarvisが利用可能なすべてのツールの中央カタログです。複数のソースからツールを集約し、統一されたインターフェースを通じて提示します。

## ツールソース

| ソース | 説明 |
|--------|-------------|
| **ネイティブモジュール** | 組み込みCoreLayerツール（タスク管理、リーディングリスト、レビュー） |
| **MCPサーバー** | 接続されたMCPサーバーからのツール |
| **スキル** | カスタムスクリプトワークフロー |
| **RESTアダプター** | REST APIエンドポイントからのツール |

## 仕組み

```text
ツールソースがツールを登録
  → ツールレジストリがすべてのツールをカタログ化
  → Jarvisがリクエストに対して利用可能なツールをクエリ
  → ツールレジストリが一致するツールを返却
  → Jarvisが適切なツールを選択して呼び出し
  → パーミッションガードが実行前にチェック
```

## ツールメタデータ

レジストリ内の各ツールには以下が含まれます:

- **名前** — 一意の識別子
- **説明** — ツールの機能
- **パラメータ** — 入力スキーマ（JSON Schema）
- **ソース** — このツールを所有するプロバイダー
- **リスクレベル** — read、write、external、destructive

## ツールのクエリ

Jarvisはレジストリにクエリしてツールを検索できます:

- **機能別** — "ファイルを読み取れるツールを見つけて"
- **ソース別** — "すべてのMCPツールを表示"
- **名前別** — "filesystem:readツールを使用して"
