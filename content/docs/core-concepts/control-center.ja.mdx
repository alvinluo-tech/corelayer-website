---
title: コントロールセンター
description: モデル、ツール、権限、システムヘルスを管理するための中央UI。
---

コントロールセンターはCoreLayerのメインインターフェースです。システムのすべての側面を管理する統一されたサーフェスを提供します。

## パネル

| パネル | 用途 |
|-------|---------|
| **Chat** | Jarvisとのメイン会話 |
| **Models** | モデルプロバイダーの設定と監視 |
| **MCP** | MCPサーバー接続の管理 |
| **Tools** | 登録されたツールとそのステータスの表示 |
| **Permissions** | 承認キューと監査ログ |
| **Voice** | 音声パイプラインのステータスと設定 |
| **Settings** | システム全体の設定 |

## コマンドパレット

**Alt+Space**でコマンドパレットを開き、クイックアクションを実行:

- 会話を検索
- ツールを直接実行
- パネル間を切り替え
- 設定をクイック設定

## ステータスバー

コントロールセンターの下部に表示:

- デーモンのヘルスステータス
- 接続されたMCPサーバー数
- アクティブなモデルプロバイダー
- 音声パイプラインの状態
- 保留中の承認数

## レイアウト

コントロールセンターはサイドバーレイアウトを使用:

- **左サイドバー** — パネル間のナビゲーション
- **メインエリア** — アクティブなパネルのコンテンツ
- **右パネル** — コンテキスト依存の詳細（ツール情報、承認詳細）
