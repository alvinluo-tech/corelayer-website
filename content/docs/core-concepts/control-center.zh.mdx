---
title: 控制中心
description: 管理模型、工具、权限和系统健康状况的中心 UI。
---

控制中心是 CoreLayer 的主界面。它提供了一个统一的界面来管理系统的所有方面。

## 面板

| 面板 | 用途 |
|------|------|
| **聊天** | 与 Jarvis 的主要对话 |
| **模型** | 配置和监控模型提供商 |
| **MCP** | 管理 MCP 服务器连接 |
| **工具** | 查看已注册工具及其状态 |
| **权限** | 审批队列和审计日志 |
| **语音** | 语音管线状态和配置 |
| **设置** | 系统级配置 |

## 命令面板

按 **Alt+Space** 打开命令面板进行快速操作：

- 搜索对话
- 直接执行工具
- 在面板之间切换
- 快速配置设置

## 状态栏

控制中心底部显示：

- 守护进程健康状态
- 已连接的 MCP 服务器数量
- 活跃的模型提供商
- 语音管线状态
- 待审批数量

## 布局

控制中心使用侧边栏布局：

- **左侧侧边栏** — 面板间导航
- **主区域** — 当前面板内容
- **右侧面板** — 上下文相关的详情（工具信息、审批详情）
