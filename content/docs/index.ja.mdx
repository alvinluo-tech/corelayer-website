## CoreLayerとは？

CoreLayerは、あなたとパーソナルアプリエコシステムの間に位置します。ブラウザタブに閉じ込められたチャットボットではなく、デスクトップ制御レイヤーとして動作します:

- **Voice-native** — ウェイクワード、ストリーミングASR/TTS、バージイン割り込み
- **MCP-first** — Model Context Protocolを通じて外部ツールサーバーを接続
- **Tool-aware** — ネイティブモジュール、MCPツール、スキル、RESTアダプターの統一レジストリ
- **Permission-guarded** — リスクのあるアクションは分類され、承認待ちになり、ログに記録されます
- **Local-first** — ローカルのSQLite、オプションのSupabaseクラウド同期
