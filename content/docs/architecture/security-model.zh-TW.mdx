CoreLayer 採用本地優先、權限控制的安全模型設計。

## 核心原則

1. **本地優先** — 資料預設儲存在你的機器上
2. **權限控制** — 工具執行需要明確的策略
3. **可稽核** — 所有操作都會被記錄
4. **可見** — 沒有任何事情會在你不知情的情況下發生

## 資料安全

### 本地儲存

預設情況下，所有資料儲存在本地 SQLite 資料庫中：

- 對話
- 任務和閱讀清單
- 稽核日誌
- 模型設定
- 使用者偏好設定

資料庫檔案儲存在使用者的應用程式資料目錄中，不會傳輸到任何地方。

### 可選的雲端同步

啟用 Supabase 時：

- 資料在傳輸中加密（HTTPS）
- Supabase 處理靜態加密
- 你控制 Supabase 專案及其存取策略

### API 金鑰

- 儲存在本地設定檔中
- 可以引用環境變數
- 永遠不會傳輸到 CoreLayer 自己的伺服器
- 僅傳送到已設定的 AI 供應商

## 工具執行安全

### 權限守護

每個工具呼叫都經過權限守護：

```text
工具呼叫 → 風險分類 → 策略檢查 → 執行或封鎖
```

### 風險等級

| 等級 | 自動執行？ | 範例 |
|-------|---------------|----------|
| 讀取 | 是 | 列出檔案、讀取資料 |
| 寫入 | 否（確認） | 建立/修改檔案 |
| 外部 | 否（確認） | API 呼叫 |
| 破壞性 | 否（確認 + 警告） | 刪除資料 |
| 憑證敏感 | 否（封鎖） | 存取權杖 |

### 稽核軌跡

每次工具執行都會記錄：

- 時間戳記
- 工具名稱和來源
- 參數（已清理）
- 風險等級
- 使用者決定
- 執行結果

## 網路安全

### 僅限本機

守護程式 API 僅可從本機存取：

- 不暴露到網路
- 預設無遠端存取
- Tauri IPC 僅限本機

### 外部連接

CoreLayer 僅連接到：

- 你設定的 **AI 供應商**（Groq、OpenRouter 等）
- 你新增的 **MCP 伺服器**
- 如果你啟用雲端同步，則連接到 **Supabase**

無遙測、分析或回報連接。

## Tauri 安全

Tauri 提供額外的安全保障：

- **沙盒化 webview** — 前端不能直接存取系統
- **IPC 白名單** — 僅允許已註冊的指令
- **程序隔離** — 守護程式作為獨立程序運行

## 威脅模型

| 威脅 | 緩解措施 |
|--------|-----------|
| 惡意工具呼叫 | 權限守護 + 稽核日誌 |
| 資料外洩 | 本地優先儲存，無遙測 |
| 未授權存取 | 僅限本機 API |
| 模型提示注入 | 工具參數驗證、風險分類、非讀取呼叫的權限守護確認 |
| MCP 伺服器被入侵 | 權限守護封鎖有風險的呼叫 |
