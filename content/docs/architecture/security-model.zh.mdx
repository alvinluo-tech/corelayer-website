description: CoreLayer 如何保护你的数据并控制工具执行。

CoreLayer 采用本地优先、权限控制的安全模型设计。

## 核心原则

1. **本地优先** — 数据默认存储在你的机器上
2. **权限控制** — 工具执行需要明确的策略
3. **可审计** — 所有操作都被记录
4. **可见** — 没有任何操作会在你不知情的情况下发生

## 数据安全

### 本地存储

默认情况下，所有数据存储在本地 SQLite 数据库中：

- 对话记录
- 任务和阅读列表
- 审计日志
- 模型配置
- 用户偏好

数据库文件存储在用户的应用数据目录中，不会传输到任何地方。

### 可选云同步

启用 Supabase 时：

- 数据在传输过程中加密（HTTPS）
- Supabase 处理静态加密
- 你控制 Supabase 项目及其访问策略

### API 密钥

- 存储在本地配置文件中
- 可以引用环境变量
- 永远不会传输到 CoreLayer 自己的服务器
- 仅发送到配置的 AI 提供商

## 工具执行安全

### 权限守卫

每个工具调用都经过权限守卫：

```text
工具调用 → 风险分类 → 策略检查 → 执行或阻止
```

### 风险级别

| 级别 | 自动执行？ | 示例 |
|------|-----------|------|
| 读取 | 是 | 列出文件、读取数据 |
| 写入 | 否（确认） | 创建/修改文件 |
| 外部 | 否（确认） | API 调用 |
| 破坏性 | 否（确认 + 警告） | 删除数据 |
| 凭据敏感 | 否（阻止） | 访问令牌 |

### 审计追踪

每次工具执行都记录以下信息：

- 时间戳
- 工具名称和来源
- 参数（已脱敏）
- 风险级别
- 用户决策
- 执行结果

## 网络安全

### 仅限本地主机

守护进程 API 仅可从本地主机访问：

- 不暴露到网络
- 默认不允许远程访问
- Tauri IPC 仅限本地

### 外部连接

CoreLayer 仅连接到：

- 你配置的 **AI 提供商**（Groq、OpenRouter 等）
- 你添加的 **MCP 服务器**
- 如果你启用了云同步，则连接 **Supabase**

没有遥测、分析或回传连接。

## Tauri 安全

Tauri 提供额外的安全保障：

- **沙盒化 Webview** — 前端不能直接访问系统
- **IPC 白名单** — 仅允许注册的命令
- **进程隔离** — 守护进程作为独立进程运行

## 威胁模型

| 威胁 | 缓解措施 |
|------|----------|
| 恶意工具调用 | 权限守卫 + 审计日志 |
| 数据泄露 | 本地优先存储，无遥测 |
| 未授权访问 | 仅限本地主机的 API |
| 模型提示注入 | 工具参数验证、风险分类、非读取调用的权限守卫确认 |
| 被入侵的 MCP 服务器 | 权限守卫阻止高风险调用 |
