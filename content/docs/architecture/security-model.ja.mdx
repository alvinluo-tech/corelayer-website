---
title: セキュリティモデル
description: CoreLayerがデータを保護し、ツール実行を制御する方法。
---

CoreLayerは、ローカルファーストで権限制御されたセキュリティモデルで設計されています。

## コア原則

1. **ローカルファースト** — デフォルトではデータはローカルマシンに残る
2. **権限制御** — ツール実行には明示的なポリシーが必要
3. **監査可能** — すべてのアクションがログに記録される
4. **可視** — あなたの知らないうちに何も起きない

## データセキュリティ

### ローカルストレージ

デフォルトでは、すべてのデータはローカルSQLiteデータベースに保存されます:

- 会話
- タスクとリーディングリスト
- 監査ログ
- モデル設定
- ユーザー設定

データベースファイルはユーザーのアプリケーションデータディレクトリに保存され、どこにも送信されません。

### オプションのクラウド同期

Supabaseが有効な場合:

- データは転送中に暗号化される（HTTPS）
- Supabaseが保存時の暗号化を処理
- あなたがSupabaseプロジェクトとそのアクセスポリシーを制御

### APIキー

- ローカル設定ファイルに保存
- 環境変数を参照可能
- CoreLayer自体のサーバーには送信されない
- 設定されたAIプロバイダーにのみ送信

## ツール実行セキュリティ

### パーミッションガード

すべてのツール呼び出しはパーミッションガードを通過します:

```text
ツール呼び出し → リスク分類 → ポリシーチェック → 実行またはブロック
```

### リスクレベル

| レベル | 自動実行？ | 例 |
|-------|---------------|----------|
| Read | はい | ファイル一覧、データ読み取り |
| Write | いいえ（確認） | ファイル作成/変更 |
| External | いいえ（確認） | API呼び出し |
| Destructive | いいえ（確認＋警告） | データ削除 |
| Credential-sensitive | いいえ（ブロック） | アクセストークン |

### 監査証跡

すべてのツール実行が以下とともにログに記録されます:

- タイムスタンプ
- ツール名とソース
- パラメータ（サニタイズ済み）
- リスクレベル
- ユーザーの決定
- 実行結果

## ネットワークセキュリティ

### ローカルホストのみ

デーモンAPIはローカルホストからのみアクセス可能です:

- ネットワークには公開されない
- デフォルトではリモートアクセスなし
- Tauri IPCはローカルのみ

### 外部接続

CoreLayerが接続するのは以下のみです:

- あなたが設定した**AIプロバイダー**（Groq、OpenRouterなど）
- あなたが追加した**MCPサーバー**
- クラウド同期を有効にした場合の**Supabase**

テレメトリ、分析、またはファーンホーム接続はありません。

## Tauriセキュリティ

Tauriは追加のセキュリティを提供します:

- **サンドボックス化されたWebview** — フロントエンドがシステムに直接アクセスできない
- **IPCホワイトリスト** — 登録されたコマンドのみ許可
- **プロセス分離** — デーモンは別プロセスとして実行

## 脅威モデル

| 脅威 | 対策 |
|--------|-----------|
| 悪意のあるツール呼び出し | パーミッションガード＋監査ログ |
| データ流出 | ローカルファーストストレージ、テレメトリなし |
| 不正アクセス | ローカルホスト限定API |
| モデルプロンプトインジェクション | ツールパラメータ検証、リスク分類、非Read呼び出しに対するパーミッションガードの確認 |
| 侵害されたMCPサーバー | パーミッションガードがリスクのある呼び出しをブロック |
