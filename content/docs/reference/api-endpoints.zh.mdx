CoreLayer 守护进程在 `localhost:3001` 上暴露 HTTP API。Tauri 前端通过 Tauri 的 IPC 桥接使用此 API。

## 基础 URL

```text
http://localhost:3001
```

## 聊天

### POST `/api/chat`

向 Jarvis 发送消息并接收流式响应。

```json
{
  "message": "What tasks do I have?",
  "conversationId": "conv_123"
}
```

响应：服务器发送事件（SSE）流。

## 任务

### GET `/api/tasks`

列出所有任务。

```json
// 响应
[
  {
    "id": "task_abc123",
    "title": "Review PR #42",
    "status": "pending",
    "priority": "high",
    "dueDate": "2026-06-30T00:00:00Z",
    "createdAt": "2026-06-28T10:00:00Z"
  }
]
```

### POST `/api/tasks`

创建新任务。

```json
// 请求
{
  "title": "Review PR #42",
  "priority": "high",
  "dueDate": "2026-06-30"
}

// 响应
{
  "id": "task_abc123",
  "title": "Review PR #42",
  "status": "pending",
  "priority": "high",
  "dueDate": "2026-06-30T00:00:00Z"
}
```

### PATCH `/api/tasks/:id`

更新任务。接受部分任务对象。

### DELETE `/api/tasks/:id`

删除任务。

## 文章（阅读列表）

### GET `/api/articles`

列出所有文章。

### POST `/api/articles`

添加文章。

### DELETE `/api/articles/:id`

移除文章。

## 回顾

### GET `/api/reviews`

列出回顾。

### POST `/api/reviews/generate`

生成新的回顾。

## 模型

### GET `/api/models`

列出已配置的模型提供商。

### POST `/api/models`

添加模型提供商。

## MCP

### GET `/api/mcp/servers`

列出 MCP 服务器。

### POST `/api/mcp/servers`

添加 MCP 服务器。

### POST `/api/mcp/servers/:id/restart`

重启 MCP 服务器。

## 工具

### GET `/api/tools`

列出所有已注册的工具。

## 权限

### GET `/api/permissions/audit`

获取审计日志条目。

### POST `/api/permissions/approve`

批准待处理的工具调用。

### POST `/api/permissions/deny`

拒绝待处理的工具调用。

## 健康检查

### GET `/api/health`

检查守护进程健康状况。

```json
{
  "status": "healthy",
  "uptime": 3600,
  "version": "<current-version>"
}
```

## 身份验证

守护进程 API 仅可从本地主机访问。本地访问无需身份验证。
