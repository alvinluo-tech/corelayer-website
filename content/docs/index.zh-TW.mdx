## 什麼是 CoreLayer？

CoreLayer 位於你和你的個人應用程式生態系統之間。它不是被困在瀏覽器分頁中的聊天機器人，而是作為桌面控制層運作：

- **語音原生** — 喚醒詞、串流 ASR/TTS、打斷式插話
- **MCP 優先** — 透過模型上下文協定連接外部工具伺服器
- **工具感知** — 原生模組、MCP 工具、技能和 REST 轉接器的統一註冊表
- **權限守護** — 有風險的操作會被分類、暫停等待核准，並記錄日誌
- **本地優先** — 你機器上的 SQLite，可選的 Supabase 雲端同步
