CoreLayerは、Jarvisを搭載したデスクトップAIコマンドレイヤーです。汎用チャットボットでも、IDEコパイロットでも、クラウドオートメーションツールでもありません。ローカルで個人のアプリ、ツール、モデル、MCPサーバーの間に位置し、Jarvisが明示的な権限を通じて作業を調整します。

## 主要機能

- **マルチモデルルーティング** — MiMo、Groq、OpenRouter、Ollama、OpenAI互換プロバイダー間でルーティング
- **MCP-first統合** — 外部ツールおよびサーバーとの統合
- **統一ツール呼び出し** — ネイティブモジュール、MCP、スキル、RESTアダプターからの呼び出し
- **パーミッションガード** — リスクベースの実行と監査ログ
- **音声パイプライン** — ウェイクワード、ASR、ストリーミングTTS、バージイン割り込み
- **3つのストレージモード** — ローカルSQLite、Supabase、PostgreSQL（ランタイム中にホットスワップ可能）
- **コマンドパレット**（Alt+Space） — クイックアクション
- **タスク管理** — AI搭載の自然言語クエリ
- **リーディングリストトラッキング** — AIレコメンデーション付き
- **日次/週次レビュー要約** — パターン認識付き

## アーキテクチャ概要

```text
Tauri 2デスクトップアプリ
  └─ React 19 + Vite + Tailwind 4フロントエンド
     └─ Tauri IPCブリッジ
        └─ Node.jsデーモン（Hono）
           ├─ モデルゲートウェイ（Vercel AI SDK）
           ├─ ツールレジストリ
           ├─ パーミッションガード
           ├─ MCPクライアントマネージャー
           └─ SQLite / Supabase / PostgreSQL
```

CoreLayerは三層プロキシモデルを使用: ReactフロントエンドはTauri IPCを通じてNode.jsデーモンと通信し、デーモンがすべてのAIオーケストレーション、ツール実行、データストレージを管理します。

## CoreLayerのターゲットユーザー

- AIワークフローをローカルで制御したい**技術ユーザー**
- MCPとローカルAIツールで構築する**インディー開発者**
- マルチモデルルーティングと音声インターフェースを試す**パワーユーザー**
- オープンソースデスクトップAI分野に関心を持つ**将来のコントリビューター**
